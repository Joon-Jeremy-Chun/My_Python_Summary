# HTML5 기본구조

+ `<>`테그라 부른다. (기본 블럭)
+ `<요소> 내용 </요소>` 형식이다.
+ `<html>` 기본시작
+ html -> head -> body 의 기본형식
+ 저장을 `.html`


## 내부 시트일시트 

CSS

1. 내부에 css 를 직접 작성해 사용 한다.
2. 외부내용(저장한 내용 `.css`)을 연결해서 사용 할 수 있다.

Javascript

1. 내부에 작정 가능`<script> </script>`
2. 외부내용(저장한 내용 `.js`)을 연결해서 사용 할 수 있다.
